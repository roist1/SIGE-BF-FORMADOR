# SIGE-BF-FORMADOR

## Sistema de Apoio ao Formador de Bombeiros

Projeto académico desenvolvido no âmbito da disciplina de
Fundamentos e Conceção de Sistemas de Informação.

### Autores

- Sérgio Simões
- Ricardo Reis

### Objetivo

Criar um sistema de apoio ao formador de bombeiros, integrando
conteúdos formativos, simulação conceptual, documentação técnica
e modelação UML.

### Estrutura

- /docs → Relatório académico
- /formacao → Conteúdos programáticos
- /diagrams → UML
- /slides → Defesa final
